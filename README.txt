EXCEL FINANCIAL PLANNING CODE

This will be a suite of python code that will integrate with excel to form a comprehensive set of finanical planning tools.

Planned capabilites:

- Pull financial account information from an excel sheet

- Import OFX data from the internet based on accounts specified in the spreadsheet

- Parse OFX data and populate the spreadsheet with the information

- Import stock data to show the performance of my portfolia

- Analyze investment oppurtunites

In general the goal of this project is to leverage the convienant graphical user interface and graphing tools of excel with the advanced data manipulation capabilities of python and its libraries. XLWings is the planned tool that will integrate Python with excel.
